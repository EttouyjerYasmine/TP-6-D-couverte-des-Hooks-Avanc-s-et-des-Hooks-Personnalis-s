# TP Hooks Débutant - React

## Description
Ce projet est un mini TP React pour pratiquer les **Hooks** et la **gestion d'état**.  
Il contient :  

- Un **compteur** avec incrément, décrément et remise à zéro.  
- Sauvegarde automatique dans **localStorage**.  
- Une **horloge** affichant l'heure actuelle mise à jour toutes les secondes.  
- Utilisation des hooks **useState**, **useReducer**, **useEffect** et **useRef**.
